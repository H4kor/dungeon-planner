# Installation

## Ubuntu (Linux)

- Download the [latest release from Github](https://github.com/H4kor/dungeon-planner/releases/latest)
- allow execution of the downloaded binary `chmod +x dungeon_planner_x86_64`
- start with `./dungeon_planner_x86_64`
